CSDC GenAI Hub

Project Title:
CSDC GenAI Hub – Learn About Generative AI, Ethics, and Real-World Examples

Short Overview:
Welcome to the CSDC GenAI Hub! This website is designed to help people understand how generative tools work, their strengths and limitations, and how to use them responsibly.

The website includes:
- A FAQs section with collapsible questions for easy reading
- An Ethics section explaining responsible use of technology
- An Article section with real-life analogies and insights
- Clickable references to reliable sources for further learning

The website is fully responsive and looks great on phones, tablets, and desktops.

How to Run the Project:
1. Download or copy the project folder
2. Make sure all HTML files (index.html, FAQs.html, Ethics.html, Article.html) are in the same folder
3. Open index.html in your preferred web browser (Chrome, Firefox, Edge, etc.)
4. That’s it! No extra setup or server is needed — you’re ready to explore

Frameworks and Libraries Used:
- Bootstrap 5.3 for responsive layout, navigation, collapsible sections, and buttons
- Custom CSS (effect.css) for hover-glow effects and styling
- Bootstrap JavaScript bundle for interactive collapsible sections and buttons
